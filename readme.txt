Quartz Plugin lets you show random quotes/tips/links/pictures/something to visitors. These quotes can be added from the admin side.

== Description ==

You can add the quotes one by one - or you can bulk import the stuff from a text file.	

== Installation ==

1. Download the zipped file.
1. Extract and upload the contents of the folder to /wp-contents/plugins/ folder
1. Go to the Plugin management page of WordPress admin section and enable the 'Quartz' plugin
1. Go to Manage > Manage Quotes to add/edit quotes
1. Add the code <?php quartz_show() ?> in your template - this will dispaly the quote at that location.
